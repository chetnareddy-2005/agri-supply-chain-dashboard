# 🌾 Agri Supply Chain Dashboard

## 📌 Overview

A full-stack web dashboard connecting farmers and retailers for efficient supply chain management and real-time insights.

## 🚀 Features

* Farmer product listing
* Retailer order management
* Secure authentication
* Payment integration (Cashfree)
* Dashboard analytics

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Spring Boot (Java)
* Database: MySQL
* Payment: Cashfree

## ⚙️ How to Run

### Backend

cd backend
mvn spring-boot:run

### Frontend

cd frontend
npm install
npm start

## 🔐 Environment Variables

Set these before running:

CASHFREE_APP_ID=your_app_id
CASHFREE_SECRET_KEY=your_secret_key
